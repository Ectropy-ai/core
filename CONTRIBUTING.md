# Contributing to Ectropy Core

Ectropy Core is maintained by [LuhTech Holdings](https://luh.tech).

**We are not accepting external contributions at this time.**

You are welcome to:
- Open issues to report bugs or request features
- Fork the repository for your own use under the Apache-2.0 license

When we begin accepting contributions, this document will be
updated with CLA requirements and submission guidelines.

Questions? Contact hello@luh.tech
